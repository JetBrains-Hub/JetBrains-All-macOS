## What's inside (instant developer happiness)

- `Themes/` — 500+ custom dark/light themes (Dracula Pro, One Dark, Material Oceanic, Nord, etc.)  
- `Keymaps/` — VS Code, Sublime, Vim, IntelliJ + hybrid layouts  
- `Live Templates/` — 10 000+ snippets (React, Flutter, Spring, Django, Laravel, etc.)  
- `Plugin Bundles/` — pre-selected must-have plugins for each IDE  
- `Code Styles/` — Airbnb, Google, PSR-12, Kotlin, Rust ready  
- `Toolbox Configs/` — auto-updates disabled, memory settings, JVM tweaks  
- `Font Packs/` — JetBrains Mono + Fira Code + Cascadia Code ligatures  
- `Shell Scripts/` — one-click import for all IDEs

## Installation (macOS) – 60 seconds

1. Click the button → downloads  
2. Run `install.sh` or drag folders into `~/Library/Application Support/JetBrains`  
3. Restart any JetBrains IDE → everything loads automatically
